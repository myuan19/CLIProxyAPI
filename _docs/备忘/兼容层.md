目前兼容层就是负责做请求体格式和端点不一样的时候的转换的吧？

对，就是这样。

**兼容层（AutoCompat）的唯一职责**：当请求体格式 ≠ 端点期望格式时，把请求体转换成端点格式。

```
客户端发 Claude 格式 body → /v1/chat/completions (期望 OpenAI)
                                    │
                            AutoCompat 检测到不匹配
                            Claude → OpenAI 转换
                                    │
                                    ▼
                    后续所有环节拿到的都是 OpenAI 格式 body
```

格式一致或无法识别时，直接放行，什么都不做。
